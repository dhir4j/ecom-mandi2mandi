# **App Name**: Mandi2Mandi

## Core Features:

- Product Listing: Display a list of agricultural products with images, names, locations, and price ranges in a card/grid format.
- Order Form: Show an order form when the user clicks 'Buy Now', pre-filled with the product name and fields for quantity, unit, buyer name, mobile number, and address.
- Dynamic Pricing: Implement dynamic price calculation based on the selected payment type (100% Advance or 50% Advance with a 10% increase).
- Checkout Page: Display a checkout page with a product summary and a UPI QR code for payment, along with an input field for the 12-digit UTR code.
- Order Confirmation: Show a confirmation message after submitting the UTR code and a 'Back to Home' button.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to reflect the agricultural nature of the platform.
- Background color: Light desaturated green (#F0FFF0) to create a clean and trustworthy feel.
- Accent color: Muted yellow (#BDB76B) for buttons and highlights, drawing inspiration from harvested crops.
- Body and headline font: 'PT Sans' for a modern and readable style.
- Use simple, clean icons representing different product categories and actions (e.g., cart, location, payment).
- Employ a clean, card-based layout similar to Bijak Mandi to provide a familiar and trustworthy experience.
